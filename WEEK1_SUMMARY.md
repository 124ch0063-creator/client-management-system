## 🎨 Design Specifications

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Blue | #3498db | Buttons, links, primary actions |
| Dark Blue | #2c3e50 | Text, headers, main elements |
| Success Green | #2ecc71 | Success state, closed items |
| Warning Orange | #f39c12 | Warning state, in-progress items |
| Danger Red | #e74c3c | Error state, open items |
| Gray | #7f8c8d | Secondary text, labels |

### Typography
- **Headings:** 24-32px, Bold, Dark Blue
- **Subheadings:** 18-20px, Bold
- **Body Text:** 14-16px, Regular
- **Labels:** 14px, Bold
- **Buttons:** 14-16px, Bold

### Spacing System
- Base unit: 8px
- Padding: 12px, 15px, 20px, 30px
- Margins: 15px, 20px, 30px, 40px
- Gap: 15px, 20px
